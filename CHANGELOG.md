# Changelog

## v1.2.0 — 动态需求 + 像素角色
- 🎲 新增四种动态季节性需求模式（夏季旺季・冬季旺季・双峰・突发冲击）
- 🎭 新增四个像素风格角色（CSS pixel art，带弹跳动画）
- 📊 复盘页面揭露本局需求模式
- 💡 游戏中需求趋势指示（↑上升 / ↓下降）

## v1.1.0 — 游戏核心完成
- ⚡ WebSocket 实时四人多人对战
- 🔒 信息隔离（服务器端保证，每人只收到自己角色数据）
- ⏱️ 每周 60 秒计时，时间到自动补齐
- 📈 Chart.js 历史图表（库存趋势・订单趋势）
- 🏁 复盘页面（牛鞭效应全景图・成绩排名）

## v1.0.0 — 初始版本
- 🏗️ React + Vite 前端架构
- 🖥️ Node.js + Express + Socket.io 后端
- 🗂️ 内存房间管理（无需数据库）
- 🎮 四个角色：零售商・批发商・分销商・制造商
- 📝 角色卡片（胜利条件・信息权限・策略提示）
