# Demo 單機版

## 使用方式

直接用瀏覽器打開 `beer_game_demo.html` 即可遊玩，無需安裝任何工具。

## 特色

- **單人模式**：你選擇一個角色，其他三個角色由 AI 自動扮演
- **完整遊戲流程**：選角色 → 角色介紹 → 20週遊戲 → 複盤
- **Pixel Art 風格**：Cubic 11 中文像素字體 + Canvas 繪製角色
- **動態需求**：每局隨機生成週期性需求曲線
- **複盤揭曉**：遊戲結束才告訴你是哪種需求模式

## 與多人版本的差異

| 功能 | Demo 單機版 | 多人連線版 |
|------|------------|----------|
| 人數 | 1人 | 2-4人 |
| 其他角色 | AI 自動 | 真人玩家 |
| 資訊隔離 | 模擬 | 真實（服務器端）|
| 安裝需求 | 無 | Node.js |
| 網路需求 | 無 | 需要 |

## AI 策略

其他三個 AI 角色使用簡單的目標庫存策略：

```
下單量 = max(0, 目標庫存 - 當前庫存 - 在途貨物 + 積壓訂單)
目標庫存 = 14 箱（固定）
```

這個策略比完全隨機聰明，但仍會有牛鞭效應，讓你可以觀察現象。
