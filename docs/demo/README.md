# Demo — 啤酒供應鏈遊戲

單一 HTML 檔案，無需安裝，直接瀏覽器打開即可遊玩。

## 線上體驗

**GitHub Pages：** https://royhsu1012.github.io/beer-game/

## 使用方式

直接打開 `docs/beer_game_demo.html`，或部署至任何靜態檔案服務。

## 模式說明

### 多人連線模式
- 連線至 Render 後端（`SERVER_URL` 需正確設定）
- 支援 1-4 名真人玩家
- 不足 4 人可加入 Bot 填補任意角色
- 房主可在等待室新增/移除 Bot
- 支援斷線重連（60 秒內）

### 單人模式
- 完全離線，無需後端
- 你選擇一個角色，其他三個由 AI 自動扮演
- AI 使用目標庫存策略：`下單量 = max(0, 目標庫存 - 庫存 - 在途 + 積壓)`

## 各角色定價與成本

| 角色 | 賣價 | 進貨/生產 | 毛利率 | 持有費 | 缺貨罰 |
|------|------|-----------|--------|--------|--------|
| 🏪 零售商 | $30 | $18 | 40% | $3/週 | $6/週 |
| 📦 批發商 | $20 | $13 | 35% | $2/週 | $4/週 |
| 🚛 分銷商 | $14 | $10 | 29% | $2/週 | $4/週 |
| 🏭 製造商 | $10 | $8 | 20% | $1/週 | $2/週 |

- 起始資金：$2000
- 期初庫存：12 箱，期初在途：[0, 0]

## Bot 策略

```
下單量 = max(0, 平均近3週需求 × 1.5 - 當前庫存 - 在途 + 積壓 + 隨機噪音[-2,+2])
```

策略比完全隨機聰明，但仍會產生牛鞭效應，讓你觀察現象。

## 技術說明

- 純 HTML/CSS/JavaScript，零依賴框架
- Socket.io CDN 引入（多人模式）
- Canvas 繪製 Pixel Art 角色
- Cubic 11 中文像素字體
- Web Audio API 倒數計時提示音
- 成本表與任務文字由 JS 常數動態生成（修改一處全自動同步）
